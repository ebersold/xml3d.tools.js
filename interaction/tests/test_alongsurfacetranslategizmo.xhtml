AlongSurfaceTranslateGizmo

0.8 0 0 1 0.8 0.8 0.8
With this gizmo an object can be translated on the surface it stands on. In the back there is a step. If you move the object towards it at some point it will stand upon the step. Also the whole surface itself is tilted a little bit.
Controls:
  • left mouse-button: activate the dragging operation
  • Ctrl while dragging: move the object in the horizontal plane ignoring the surface
