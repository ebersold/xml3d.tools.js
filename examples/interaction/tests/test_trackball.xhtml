Upon activation the rotation of the rectangle can be controlled by dragging it. This is done using a TrackBall. A TrackBall delivers the rotation behavior given a mouse position. It is the base class for the Rotater.
The status can be reviewed below.
  • Dragging: whether the mouse is dragging and thus rotating.
  • Rotation: the rotation attribute of the target element's transform node.
